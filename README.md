# AdMore-Android-Demo

#### 介绍
[AdMore](http://admore.web.plutus-cat.com/)是一款专业的广告聚合解决方案，可以助力开发者快速实现多广告平台变现，提升ECPM。本仓库主要为Android端Demo，目的为帮助开发者快速接入SDK。

#### 接入
